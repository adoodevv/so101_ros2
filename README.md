# so101_ros2 #

![OS](https://img.shields.io/ubuntu/v/ubuntu-wallpapers/noble)
![ROS_2](https://img.shields.io/ros/v/jazzy/rclcpp)

This package contains the code for controlling a robotic arm using ROS 2. It includes the necessary nodes, messages, and services to interface with the robotic arm hardware and perform various operations such as moving the arm, controlling the gripper, and reading sensor data.
